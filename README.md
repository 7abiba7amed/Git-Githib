# Git-Githib

```git branch -d <branch_Name>```
Delete branch locally

```git push origin :<branch_name>``` 
Delete branch remotely

```git tag```
List all tags

```git tag -d <tag_name>``` 
Delete tag locally

```git push --delete origin <tagname>```
Delete tag remotely
